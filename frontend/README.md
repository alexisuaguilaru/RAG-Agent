## Frontend -- Chat UI
Based on the project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). This folder contains the source code for the chat UI built wiht Next.js (React) for only one user without an admin panel.

### Getting Started
First, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
